# 项目简介

## 核心功能
- 基于YOLOv8的人行道信号灯检测系统，支持red和green两类信号灯识别
- 提供图片、视频及摄像头三种检测方式
- 支持批量检测功能，可同时查看所有检测结果
- 界面实时显示目标位置、目标总数和置信度信息
- 支持自定义置信度和IOU阈值设置
- 提供多种模型选择选项
- 支持检测结果保存功能
- 提供完整的数据集（9000张标注图片）
- 包含训练好的模型及训练结果
- 提供完整的项目运行教程

## 适用场景
- 智能交通监控系统
- 行人过街安全辅助系统
- 交通信号灯状态实时监测
- 计算机视觉教学与研究
- 智能驾驶环境感知系统

## 技术特点
- 采用Pycharm/VSCode开发环境
- 基于Python 3.9实现
- 集成OpenCV-Python、PyQt5等库
- 使用Ultralytics和PyTorch框架
- 提供完整UI界面和图标资源
- 包含完整的源代码和项目文件
